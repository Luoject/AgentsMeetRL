# 向 AgentsMeetRL 列表添加仓库

把用户指定的开源仓库按本仓库 taxonomy 写入 `README.md`（必要时同步 `index.html`）。

## 输入

用户会在命令后给出：GitHub URL、分类建议、论文链接、备注。缺省字段自行检索补全。

## 要求

1. 先确认是否已存在同名/同 URL 条目，避免重复。
2. 按现有分类表格式追加：`Github Repo | Stars badge | Date | Org | Paper Link`。
3. Agent 收录标准：至少具备多轮交互或工具使用（含 TIR）之一。
4. 尽量补全：RL 框架、算法、奖励类型、环境依赖（写入对应 `<details>` 技术细节，风格与邻行一致）。
5. 更新页顶分类 badge 计数（若本次增减条目）。
6. 日期格式与表内一致（如 `2026.3`）；无论文写 `--`。
7. 用简体中文说明：归入哪一类、为何符合收录标准、改了哪些行。
